LIFF 入口流程

這個頁面會：初始化 LIFF → 取得使用者 → 寫入 localStorage → 跳轉到 C3 遊戲。
  • 1 LIFF 初始化
  • 2 取得使用者資料（暱稱 / userId）
  • 3 寫入 localStorage
  • 4 跳轉到 C3 網頁
🔧 要修改的地方：
1) LIFF ID 在下面程式碼的 // TODO: YOUR_LIFF_ID
2) C3 網址在 // TODO: C3_GAME_URL（預設用同網域的 /game-c3/）
若你在一般瀏覽器直接打開測試（非 LINE 內），也會寫入 localStorage 並進行跳轉，方便開發。
本頁會在 localStorage 寫入鍵：line_name、line_userId、line_picture、from_liff
